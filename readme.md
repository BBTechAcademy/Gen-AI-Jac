# To-Do List Manager

A simple command-line to-do list application that helps you manage daily tasks.

## Features

- Add new tasks
- View all tasks
- Mark tasks as completed
- Persistent storage (saves to JSON file)

## How to Run

### 1. Python

```bash
python main.py
```

### 2. Jac

```bash
jac main.jac
```
